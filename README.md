# ChatGPT-like Chatbot

A Python chatbot using OpenAI's GPT-4o-mini model.  
This project demonstrates LLM integration, conversation memory, and terminal-based chat.

## Features
- Interactive chat in terminal
- Conversation context remembered
- Configurable temperature for creativity
